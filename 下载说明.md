# 📥 下载说明

## 文件夹内容

这是一个完整的个人博客网站，包含以下内容：

### 核心文件（必需）
- `index.html` - 首页
- `about.html` - 关于我
- `portfolio.html` - 作品展示
- `books.html` - 最近在看的书
- `blog.html` - 博客列表
- `blog-detail.html` - 博客详情
- `travel.html` - 旅行足迹
- `contact.html` - 联系我

### 资源文件（必需）
- `css/` - 样式文件夹
- `js/` - JavaScript 文件夹
- `images/` - 图片资源文件夹

### 启动脚本（可选）
- `start.sh` - macOS/Linux 启动脚本
- `start.bat` - Windows 启动脚本

### 文档文件（可选）
- `README.md` - 使用说明
- `GET_STARTED.md` - 快速开始指南
- `QUICK_START_COVERS.md` - 添加书籍封面指南
- `下载说明.md` - 本文档

## 🚀 快速开始

### 1. 启动网站

#### Windows
双击 `start.bat`

#### macOS/Linux
```bash
chmod +x start.sh
./start.sh
```

#### 或使用 Python
```bash
python -m http.server 8000
```

### 2. 访问网站

在浏览器中打开：`http://localhost:8080`

## 📚 可选：添加书籍封面

书单页面包含 4 本书，默认显示占位符。如需添加真实封面：

1. 下载 4 张封面图片（建议从豆瓣读书、京东图书）
2. 重命名为：
   - `pyramid-principle.jpg`（金字塔原理2）
   - `crucial-conversations.jpg`（关键对话）
   - `atomic-habits.jpg`（原子习惯）
   - `getting-things-done.jpg`（搞定）
3. 放入 `images/book-covers/` 目录
4. 刷新浏览器

详细说明：查看 `QUICK_START_COVERS.md`

## 🌐 部署到服务器

### GitHub Pages
1. 创建 GitHub 仓库
2. 上传所有文件
3. Settings → Pages → 选择分支
4. 完成！

### Vercel（推荐）
1. 访问 vercel.com
2. 上传文件夹
3. 自动部署

### Netlify
1. 访问 netlify.com
2. 拖拽文件夹
3. 自动部署

详细说明：查看 `GET_STARTED.md`
