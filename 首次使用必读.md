## ✅ 正确的使用方法

### 🪟 Windows 用户：
1. 双击运行：`start.bat`
2. 浏览器会自动打开：http://localhost:8000
3. 享受带有视频背景的完整网站！

### 🍎 Mac/Linux 用户：
1. 打开终端
2. 进入文件夹：`cd beedog-website-package`
3. 运行：`./start.sh`
4. 浏览器访问：http://localhost:8000

### 💻 所有平台通用：
1. 在文件夹内打开终端
2. 运行：`python -m http.server 8000`
3. 浏览器访问：http://localhost:8000

---

## 🎬 为什么看不到视频？

- **原因**：浏览器安全策略限制 `file://` 协议下的视频自动播放
- **解决**：必须通过 HTTP 服务器运行网站
- **结果**：HTTP 协议下视频会自动正常播放

## 📦 快速部署到 GitHub Pages

想要把网站上线给别人看？3 分钟搞定！

1. 上传所有文件到 GitHub 仓库
2. Settings → Pages → 选择 main 分支 → Save
3. 等待 1-5 分钟
4. 访问你的网站链接

**详细教程**：查看 `GITHUB_PAGES_DEPLOYMENT.md`

## 🆘 遇到问题？

1. **看不到视频** → 必须通过 HTTP 服务器运行（见上方）
2. **没有 Python** → 下载安装：https://www.python.org/downloads/
3. **端口被占用** → 使用其他端口，如 `python -m http.server 8080`
4. **启动脚本无法运行** → 手动运行 `python -m http.server 8000`
