# 🚀 GitHub Pages 快速部署

## 最快部署方法（3分钟）

### 步骤 1：创建 GitHub 仓库
1. 访问 https://github.com
2. 登录后点击右上角 "+" → "New repository"
3. 仓库名称：`beedog-website`（或您喜欢的名字）
4. 选择 **Public**（必须）
5. 点击 "Create repository"

### 步骤 2：上传文件
1. 在仓库页面，点击 "uploading an existing file"
2. 将 `beedog-website-package` 文件夹内的所有文件拖拽上传
3. 等待上传完成
4. 在底部 "Commit changes" 输入：`Initial commit`
5. 点击 "Commit changes"

### 步骤 3：启用 GitHub Pages
1. 点击仓库顶部的 "Settings"
2. 左侧菜单找到 "Pages"
3. Source 选择 "Deploy from a branch"
4. Branch 选择 `main`，Folder 选择 `/ (root)`
5. 点击 "Save"

### 步骤 4：访问网站
1. 等待 1-5 分钟
2. 刷新 Pages 页面，顶部会显示链接：
   ```
   https://your-username.github.io/beedog-website/
   ```
3. 点击链接即可访问！

## ✅ 完成！

您的网站已成功部署到 GitHub Pages，完全免费！
