⚠️ 重要提示

不要直接双击 index.html 文件！

直接打开 HTML 文件使用的是 file:// 协议，浏览器会阻止视频自动播放，导致你看不到视频背景。

必须通过 HTTP 服务器运行才能正常显示视频！

选择你的操作系统启动网站：

🪟
Windows
双击 start.bat
🍎
Mac/Linux
运行 ./start.sh

📋 详细操作步骤

Windows 用户：

1. 打开文件夹 beedog-website-package

2. 双击运行 start.bat 文件

3. 浏览器会自动打开：http://localhost:8000

Mac/Linux 用户：

1. 打开终端

2. 进入文件夹：cd beedog-website-package

3. 运行：./start.sh

4. 浏览器访问：http://localhost:8000

手动启动（所有平台）：

在文件夹内打开终端，运行：

python -m http.server 8000

💡 提示：启动后访问 http://localhost:8000 即可看到带有视频背景的网站

📦 部署到 GitHub Pages 后，视频会自动正常显示，无需启动服务器
